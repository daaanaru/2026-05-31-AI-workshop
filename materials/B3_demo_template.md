# B3: デモ発表テンプレート

> 16:30-17:15 のデモ大会で使う。
> 各チーム15分（発表10分+質疑5分）。
> Googleスライド or 画面共有で直接デモ。

---

## スライド構成（3-5枚で十分）

### Slide 1: タイトル
```
チーム名: 【              】
作ったもの: 【              】
メンバー: 【              】
```

### Slide 2: 課題（Before）
```
誰の:   【対象ユーザー】
何が:   【困っていること】
今は:   【現状どうやっている】
```

### Slide 3: デモ（画面共有）
```
→ ここで実際に動かして見せる

見せるポイント:
1. 【入力はこれ】
2. 【AIがこう処理して】
3. 【出力がこれ】
```

### Slide 4: 技術構成
```
使ったもの:
- AI: 【ChatGPT API / Claude / etc.】
- 言語: 【Python / JS / etc.】
- その他: 【連携サービス等】

工夫した点:
- 【               】
```

### Slide 5: 次にやるなら
```
今日できなかったこと:
- 【               】

明日から使うなら:
- 【               】

感想（一人一言）:
- 【               】
```

---

## デモのコツ

- **動くものを見せる** > スライドで説明する
- 失敗してもOK。「ここまでできた」を見せる
- 「AIにこう頼んだらこう返ってきた」のスクショは強い
- 15分は短い。練習1回やっておくと安心

## タイムキープ

| 経過 | やること |
|------|---------|
| 0:00 | タイトル+課題説明（2分） |
| 2:00 | デモ本番（6分） |
| 8:00 | 技術構成+感想（2分） |
| 10:00 | 質疑応答（5分） |
