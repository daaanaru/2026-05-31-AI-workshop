## ページ2: 申し込みフォーム（form.html）— 既存改修

既存のフォーム（現index.html）を`form.html`に移動し、以下を追加。

### 追加機能: 顔写真アップロード（Step 01）

Step 01（自己紹介）の「呼ばれ方」フィールドの下に追加:

- ラベル: 「プロフィール写真」
- 注記: 「当日の名札やWebページに使います。AIで作った自画像でもOKです。」
- 任意（Optional）
- UI要件:
  - ドラッグ&ドロップ対応のアップロードエリア
  - アップロード後に正方形トリミングUI（プレビュー付き）
  - トリミング後の画像を200x200pxにリサイズ
  - base64でlocalStorageに保存（フォーム送信時にJSONに含める）
  - 対応形式: JPEG, PNG, WebP
  - ファイルサイズ上限: 5MB
- デザイン: 既存フォームの「地形図+森」テーマに合わせる。アップロードエリアは破線ボーダー

### スプラッシュ画面
- 既存のスプラッシュ（夕焼け山脈のSVGアニメーション）はそのまま維持
- form.htmlの先頭に配置

## 技術的な制約

- フレームワーク不要。純粋なHTML + CSS + Vanilla JS
- ビルドツール不要
- GitHub Pagesでホスティング（静的サイト）
- 既存のCSS変数・カラーパレット・フォント設定を踏襲
- レスポンシブ対応必須（スマホでの閲覧が多い）
- 既存のstyles.css, app.js, config.jsの構造を尊重
